# Codex Agent for SillyTavern 1.12.12+

This module contains custom JS mods and optional overrides for SillyTavern.

- Place scripts in `scripts/`
- HTML overrides in `html/`
- Config files and presets in `configs/`

Symlink or modify SillyTavern's `index.html` to include Codex extensions after base scripts.
