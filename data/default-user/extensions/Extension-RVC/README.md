# Extension-RVC

Adds Realtime Voice Cloning capabilities to the Text-to-Speech module.

## Installation

Install via the "Download Extensions & Assets" menu or use the following URL to install manually:

```txt
https://github.com/SillyTavern/Extension-RVC
```

## Supported sources

* [rvc-python](https://github.com/daswer123/rvc-python)
* [SillyTavern Extras](https://github.com/SillyTavern/SillyTavern-Extras) (deprecated) - requires an `rvc` module.
